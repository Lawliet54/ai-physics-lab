AI Physics Lab - краткое описание проекта

Это учебный React/Vite проект виртуальной лаборатории по физике.
Основной язык интерфейса: казахский.
Целевая аудитория: школьники старших классов / студенты начального уровня, изучающие электричество и магнетизм.

Что есть в проекте:
- Главная страница платформы
- Список виртуальных лабораторных работ
- Интерактивная симуляция закона Ома
- Интерактивная симуляция закона Кулона
- Страница результатов и прогресса
- Встроенный AI-помощник по темам физики

Ключевые файлы:
- package.json — стек и зависимости
- index.html — HTML входная точка
- src/main.jsx — точка входа React
- src/index.css — базовые стили
- App.jsx — маршрутизация страниц
- Layout.jsx — общий layout приложения
- AIPhysicsLab.jsx — главная страница
- VirtualLabList.jsx — список лабораторий
- OhmsLawLab.jsx — симуляция закона Ома
- CoulombsLawLab.jsx — симуляция закона Кулона
- ResultsPage.jsx — результаты пользователя
- AIChatAssistant.jsx — встроенный помощник

Что специально НЕ включено:
- node_modules
- dist
- package-lock.json

Это сделано для того, чтобы архив был маленьким, но при этом полностью объяснял структуру и назначение проекта.
